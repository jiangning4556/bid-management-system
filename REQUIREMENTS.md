## 🎯 项目概述与技术架构

### 项目简介
投标管理系统是一个用于管理咨询项目、投标项目、供应商、物料和收支的企业级应用。支持从咨询立项、报价对比、中标转换到收支跟踪的全流程管理。

### 技术栈
- **后端**: NestJS 11.x + TypeORM + MySQL + JWT认证
- **前端**: Vue 3 + TypeScript + Vite 6.x + Element Plus
- **状态管理**: Pinia
- **图表**: ECharts

### 端口配置
- 后端: `http://localhost:3000`
- 前端: `http://localhost:5173`

### 默认账户
- 用户名: `admin`
- 密码: `admin123`

#### 📝 实现备注
- 时间筛选参数：`all` | `month` | `quarter` | `year`
- 应收总额 = SUM(合同金额或项目金额)
- 应付总额 = SUM(选中供应商金额)
- 净资金占用 = 未收金额 - 未付金额（正数=需垫资）
- 资金健康度判断：已收>已付=良好，已收=已付=平衡，已收<已付=紧张
