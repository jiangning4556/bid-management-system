# 📝 开发备注

## WebSocket注意事项

1. **连接管理**: 用户登录后自动连接，退出后断开
2. **房间隔离**: 每个用户加入独立房间 `user:{userId}`
3. **重连机制**: 启用自动重连，最多5次尝试
4. **心跳保持**: Socket.IO内置心跳机制

## 报表生成注意事项

1. **PDF中文支持**: 需要添加中文字体文件（待完善）
2. **大数据量**: 建议>1000条数据时分页生成
3. **字段映射**: 报表字段与数据库字段的映射关系
4. **定时任务**: 需要在app.module.ts中注册ScheduleModule（待完善）

## 待完善功能

1. 定时报表的定时任务调度器需要注册ScheduleModule
2. 报表数据查询需要根据实际数据源完善fetchData方法
3. PDF中文字体支持需要添加字体文件
4. 邮件发送功能（如需要）
5. 短信发送功能（如需要）
